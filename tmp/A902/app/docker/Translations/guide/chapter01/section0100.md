#   安装 Docker

Docker 提供了两个版本：`社区版 (CE)` 和`企业版 (EE)`。

Docker 社区版 (CE) 是开发人员和小型团队开始使用 Docker 并尝试使用基于容器的应用的理想之选。Docker CE 有两个更新渠道，即 `stable` 和 `edge`：
-   Stable 每个季度为您提供可靠更新
-   Edge 每个月为您提供新功能

Docker 企业版 (EE) 专为负责在生产环境中大规模构建、交付和运行业务关键型应用程序的企业开发和 IT 团队设计
