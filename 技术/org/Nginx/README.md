Nginx doc

### 简介
- 一款免费开源的高性能HTTP服务器及反向代理服务器产品，还提供IMAP/POP3代理服务等功能
### 适用场景
- 可以用作HTTP服务器，也可以作为反向代理服务器或者邮件服务器
- 能够快速响应静态页面的请求
- 支持FastCGI、SSL、Virtual Host、URL Rewrite、HTTP Basic Auth、Gzip等大量功能
### 组件
- Web请求处理机制
- 事件驱动模型
- 高级配置
- 时间管理
- 内存管理
