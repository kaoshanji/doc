#   使用JFC / Swing创建GUI

也被称为Swing教程

此跟踪将告诉您如何使用Swing组件为应用程序和小应用程序创建图形用户界面（GUI）。如果您想将JavaFX 集成到Swing应用程序中，请参阅 将JavaFX集成到Swing应用程序中。

-   [Swing入门](section010100.md)是一个快速入门课程。首先它给你一些关于Swing的背景。然后它会告诉您如何编译和运行使用Swing组件的程序。
-   [使用NetBeans IDE学习Swing](section010200.md)是开始使用Swing的最快和最简单的方式。本课将探讨NetBeans IDE的GUI构建器，这是一个强大的功能，可让您直观地构建图形用户界面。
-   [使用Swing组件](section010300.md)将告诉您如何使用每个Swing组件 - 按钮，表格，文本组件以及所有其他组件。它还会告诉你如何使用边框和图标。
-   [Swing中的并发性](section010400.md)讨论了并发性，因为它适用于Swing编程。包括有关事件分派线程和SwingWorker类的信息。
-   [使用其他Swing功能](section010500.md)可以告诉您如何使用操作，定时器和系统托盘; 如何与桌面类集成，如何支持辅助技术，如何打印表格和文本，如何创建启动画面以及如何在对话框中使用模态。
-   [布局容器中的组件](section010600.md)告诉您如何选择布局管理器，如何使用Java平台提供的每个布局管理器类，如何使用绝对定位而不是布局管理器，以及如何创建自己的布局管理器。
-   [修改外观](section010700.md)可以告诉您如何指定Swing组件的外观。
-   [拖放和数据传输](section010800.md)告诉您在应用程序中实施数据传输时需要了解的内容。
-   [编写事件监听器](section010900.md)会告诉您如何处理程序中的事件。
-   [执行自定义绘画](section011000.md)可为您提供有关绘制自己的Swing组件的信息。它讨论了特定于Swing组件的绘画问题，提供了绘画概念的概述以及绘制自己的自定义组件的示例
