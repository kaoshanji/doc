#  App：应用


##  目录
-   学习计划：跳出来看全景，钻进去看本质
    -   资料：收集、导图、内容
    -   计划：整体规划
    -   主题：具体某部分、相互关联、集中资料、导图
    -   日志：记录过程
    -   输出
- 为了面试 --> 刷题....内容聚集
  - 面试范围
  - 技术内容
  - 常见问题
  - 知识点相关联性
-   API工具：调试接口，生成文档
    -   前后端分离，在写完接口后测试是否能正常执行，然后还要复制生成文档，可以一次性完成
- 随身书签
  - 跨浏览器同步书签
  - 书签连接作为页面直接跳
  - 截取当前页内容作为图片(连接内容永不丢失)
