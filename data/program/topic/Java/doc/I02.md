#   虚拟机

Java虚拟机是Java平台的基础，并被移植到各种基于硬件的平台上，具有指令集并在运行时操作内存的抽象计算机器。

Java虚拟机负责Java SE平台的硬件和操作系统的独立性，编译代码的小尺寸（字节码），和平台安全性。
