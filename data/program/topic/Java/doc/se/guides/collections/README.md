#   集合框架

集合框架是一个统一的体系结构，用于表示和操作集合，使它们能够独立于其表示的细节进行操作。它减少了编程工作量，同时提高了性 它实现了不相关API之间的互操作性，减少了设计和学习新API的工作量，并促进了软件的重用。该框架基于十几个集合接口。它包括这些接口的实现和操作它们的算法。
