#   Java EE和Glassfish

Java EE提供了一个丰富的企业软件平台，是社区驱动型企业软件的标准。来自行业专家，商业和开源组织，Java用户组以及无数个人的贡献。每个版本都集成了符合行业需求的新功能，提高了应用程序的可移植性，并提高了开发人员的生产力 。

Glassfish 是 Java EE标准的实现。
