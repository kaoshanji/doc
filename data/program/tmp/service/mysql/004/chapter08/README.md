#   数据库迁移向导

MySQL Workbench提供了将ODBC兼容数据库迁移到MySQL的功能。
-   跨服务器转换（迁移）不同的数据库类型，包括MySQL。
-   转换表和复制数据，但不会转换存储过程，视图或触发器。
-   允许在迁移过程中进行定制和编辑。
-   适用于Linux，macOS和Microsoft Windows。
