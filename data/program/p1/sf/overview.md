#   Spring Framework 简介

Spring Framework 是面向`应用开发`人员在实际中`开发Java项目`的综合式框架。

模块及模块关系

![spring-overview](images/spring-overview.png)

-   核心容器：管理对象，解耦对象，解决对象之间的依赖，自动化了对象的过程
-   Aop...：把嵌套在各处具有相识的功能集中在一起
-   数据访问/集成：抽象事务和持久层，简化固定操作流程
-   Web：一套MVC框架，屏蔽了原生语法
-   Test：各种测试支持
