#   Lucene

Lucene 是一款高性能的、可扩展的信息检索(IR)工具库。

信息检索是指文档搜索、文档内信息搜索或者文档相关的元数据搜索等操作。

Lucene 能够融入到你的应用程序中，以增加搜索功能。

分析文本信息，建立索引，为搜索提供支持。
