#   sql

SQL是一种从关系型数据库生成、操作和检索数据的语言。

##  混个脸熟

### 方案语句

定义储存于数据库中的数据结构

### 数据语句

用于操作SQL方案语句所定义的数据结构

-   数据类型
    -   字符
    -   数字
    -   时间

-   查询：select、from、where
    -   单表查询
    -   多表查询
    -   子查询

-   条件
    -   过滤
    -   使用集合
    -   连接

-   格式化
    -   分组与聚集
    -   数据生成、转换和操作
    -   条件逻辑


### 事务语句

用于开始、结束和回滚事务
