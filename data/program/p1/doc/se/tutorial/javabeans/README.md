#   [JavaBeans（TM）](https://docs.oracle.com/javase/tutorial/javabeans/index.html)

JavaBeans™可以轻松重复使用软件组件。开发人员可以使用他人编写的软件组件，而无需了解其内部运作情况。

要理解为什么软件组件有用，请考虑组装汽车的工人。例如，她不是从零开始建立收音机，而是简单地获得收音机并将其与汽车的其余部分挂钩。
