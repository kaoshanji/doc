#   Spring Cloud

基于 Spring Boot 实现的微服务架构开发工具，整合了诸多被广泛实践和证明过的框架作为实施的基础部件。为微服务架构中涉及的配置管理、服务治理、断路器、智能路由、微代理、控制总线、全局锁、决策竞选、分布式会话和集群管理等操作提供了一种简单的开发方式。
