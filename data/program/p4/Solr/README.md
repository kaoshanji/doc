#   Solr

Solr 是基于 Lucene 的、可扩展的、部署待命的一种开源全文搜索引擎，他提供了开箱即用的多语种关键词搜索、分面搜索、智能匹配、内容聚类和相关度加权等核心功能。

搜索引擎是信息检索系统的一种类型，信息检索是"用户需求(行为)" -- 搜索技术(引擎) -- 信息检索(集合)"。
