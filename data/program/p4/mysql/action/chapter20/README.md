#   INFORMATION_SCHEMA表

INFORMATION_SCHEMA提供对数据库元数据的访问 ，有关MySQL服务器的信息（例如数据库或表的名称），列的数据类型或访问权限。有时用于此信息的其他术语是 数据字典和 系统目录
