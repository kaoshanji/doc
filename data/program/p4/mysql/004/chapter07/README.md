#   数据库设计和建模

建模功能使数据架构师能够可视化需求并解决设计问题，从而简化了数据库设计和维护。模型驱动的数据库设计是创建有效且性能良好的数据库的有效方法，同时提供灵活性来响应不断变化的数据需求。模型用于构建ER图和物理MySQL数据库。

MySQL Workbench为创建和操作数据库模型提供了广泛的功能，包括：
-   以图形方式创建和操作模型。
-   将实时数据库反向工程模型。
-   将模型转发给脚本或实时数据库。
-   创建和编辑表格并插入数据。
