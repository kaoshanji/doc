#   Nginx

一个高性能的HTTP和反向代理服务器，也是一个IMAP/POP3/SMTP服务器

Nginx是一款轻量级的Web 服务器/反向代理服务器及电子邮件（IMAP/POP3）代理服务器，并在一个BSD-like 协议下发行。其特点是占有内存少，并发能力强
