##  简介

提供对互联网和其他分布式系统原理的理解，提供这些系统地体系结构、算法和设计，展示他们如何满足当代分布式应用的需求。

采用自顶向下的方法，首先叙述在分布式系统设计中要解决的问题，然后，通过抽象模型、算法和对广泛使用的系统进行详细的实例研究，描述成功开发系统地方法。
