#   [泛型](https://docs.oracle.com/javase/tutorial/extra/generics/index.html)

在J2SE 5.0中引入了这种期待已久的类型系统增强，允许类型或方法在各种类型的对象上运行，同时提供编译时类型安全性。它为集合框架增加了编译时安全类型，并消除了铸造的苦差事。
