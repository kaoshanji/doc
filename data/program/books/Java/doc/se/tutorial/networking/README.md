#   [自定义网络](https://docs.oracle.com/javase/tutorial/networking/index.html)

Java平台受到高度重视，部分原因是它适用于编写使用互联网和万维网资源并与之互动的程序。实际上，与Java兼容的浏览器将Java平台的这种功能用于极端，通过互联网传输和运行小应用程序。

这条线索将带领您编写可在互联网上使用的Java应用程序和小应用程序的复杂性。
