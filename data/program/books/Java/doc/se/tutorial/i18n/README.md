#   [国际化](https://docs.oracle.com/javase/tutorial/i18n/index.html)

这条线索的教训将教你如何国际化Java应用程序。国际化的应用程序很容易根据世界各地最终用户的习俗和语言进行调整。

>   注意：  本教程跟踪核心国际化功能，这是桌面，企业和移动应用程序提供的附加功能所需的基础。有关更多信息，请参阅 Java Internationalization主页。
