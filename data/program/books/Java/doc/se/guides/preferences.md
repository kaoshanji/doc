#   核心Java首选项API

应用程序需要首选项和配置数据以适应不同用户和环境的需求。所述的java.util.prefs包提供应用程序存储和检索用户和系统参数和配置数据的方法。数据持久存储在依赖于实现的后备存储中。有两个独立的优先节点树，一个用于用户首选项，另一个用于系统首选项。
