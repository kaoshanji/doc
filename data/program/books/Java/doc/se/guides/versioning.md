#   [包版本识别](https://docs.oracle.com/javase/8/docs/technotes/guides/versioning/index.html)

版本控制功能支持包级版本控制，以便应用程序和applet可以在运行时识别特定Java运行时环境，VM和类包的版本。

[Java产品版本控制规范](https://docs.oracle.com/javase/8/docs/technotes/guides/versioning/spec/versioningTOC.html)
